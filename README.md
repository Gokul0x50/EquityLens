# üìä EquityLens

**EquityLens** is a stock analysis web application built with **Flask, Python, and Yahoo Finance APIs**.  
It empowers users to research equities with **real-time price data, fundamentals, technical indicators, sector insights, and embedded social sentiment** ‚Äî all in a clean, tab-based dashboard.

---

## üöÄ Features

- üîç **Stock Search** ‚Äì Enter any ticker to fetch live market data and historical charts.  
- üìà **Interactive Charts** ‚Äì Price and volume visualizations with trends.  
- üìä **Technical Indicators** ‚Äì Includes RSI, MACD, Moving Averages, and more.  
- üíº **Fundamental Analysis** ‚Äì View financial ratios, balance sheet data, and valuation metrics.  
- üè≠ **Sector Insights** ‚Äì Identify the sector of a stock and compare peers.  
- üí¨ **Social Sentiment** ‚Äì Embedded Twitter feed to monitor live discussions on the stock.

## ‚ö° Installation & Running the Project

1. **Open in PyCharm**  
   - Download or clone this repository:  
     ```bash
     git clone https://github.com/yourusername/equitylens.git
     cd equitylens
     ```

2. **Create Virtual Environment**  
   - In PyCharm, go to:  
     `File > Settings > Project: EquityLens > Python Interpreter`  
   - Create a new **virtual environment**.

3. **Install Requirements**  
   - Open the PyCharm terminal and run:  
     ```bash
     pip install -r requirements.txt
     ```

4. **Run the Project**  
   - In PyCharm, right-click on `app.py` ‚Üí **Run 'app'**  
   - Or run in terminal:  
     ```bash
     python app.py
     ```

5. **Access the Dashboard**  
   - Open your browser and go to:  
     ```
     http://127.0.0.1:5000/
     ```
